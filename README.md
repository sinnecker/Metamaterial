# Auxetic Unit Cell + COMSOL

Este projeto gera a geometria de uma célula unitária auxética, exporta em formato DXF e executa uma simulação mecânica no COMSOL Multiphysics.

---

# Dependências

Instale as bibliotecas Python necessárias:

pip install shapely mph

- Shapely → manipulação geométrica e operações booleanas  
- mph → interface Python para o COMSOL  

---

# COMSOL

É necessário ter o COMSOL Multiphysics:

- instalado na máquina  
- com licença ativa  
- acessível via linha de comando (ex: `comsol` no PATH)  

Sem isso, a simulação não será executada.
